## 🔑 Key Features

- **Text Hierarchy (BETA):** Paste plain text or AI-generated breakdowns to create validated, linked hierarchies instantly — with built-in templates and real-time validation.
- **One‑click decomposition:** Convert an item into a full plan with a single action.
- **Universal integration:** Use from Boards, Backlogs, Sprints, Queries and Work Item Details.
- **Process compatibility:** Works with inherited and custom process rules.
- **Flexible configuration:** Auto‑tagging, assignment rules and other settings.
- **Reliable & fast:** Optimized for speed with predictable results.
- **Keyboard friendly:** Complete keyboard support and shortcuts.
- **Theme aware:** Adapts to Azure DevOps light and dark themes.

## 🚀 Getting started

1. Install Work Item Decomposer from the [Azure DevOps Marketplace](https://marketplace.visualstudio.com/items?itemName=teociaps.work-item-decomposer).
2. Open a work item in Boards, Backlogs, Sprints, Queries or Work Item Details.
3. Right‑click (or use the "..." menu) and choose **Decompose work item**.
4. Create or paste your hierarchy and click **Save**.

For detailed documentation, see the [Wiki](https://github.com/teociaps/azdo-workitem-decomposer/wiki).

## ⚙️ Settings

Project administrators and granted users can customize behavior via **Project Settings > Work Item Decomposer > Settings**. Options include auto‑tagging, assignment rules, delete confirmations and more.

For complete settings details, see the [Project Settings](https://github.com/teociaps/azdo-workitem-decomposer/wiki/Project-Settings) in our documentation.
